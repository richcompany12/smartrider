# SmartRider

SmartRider는 배달 라이더들을 위한 웹 애플리케이션입니다. 복잡한 건물 구조를 쉽게 찾아갈 수 있도록 도와줍니다.

## 기능

- 구글 계정을 이용한 로그인
- 건물 정보 등록
- 건물 정보 검색
- 사이드 메뉴를 통한 간편한 네비게이션

## 기술 스택

- React
- TypeScript
- Firebase (인증 및 데이터베이스)
